Visualizing Object Oriented Programming

Creating a class

To create a class, click the "Create a class" button on the top.
You can create a class name, and add methods If you create a class method with the same name as a class, it will be detected as a getter/setter. You can also add an instance variable to a class.

Creating a struct

To create a struct, you need to click the "Create a struct" button on the top. You can add members to the struct, which have a datatype and a name.

Viewing the code

To view the generated code, select the language you want using the dropdown on the top. The options are C++ and Java currently. Then after selecting the language, click the "View Code" button on the top. If there is no code to generate, it will say "No code yet."If there is code to be generated, it will be shown.
